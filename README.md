## 포트폴리오 개요
- **개발자 :** 홍윤표
- **설명 :** 유니티의 2D 물결 효과를 구현한 스크립트 입니다.
- **사용방법 :** 빈 오브젝트를 생성하고, Water 컴포넌트를 적용해주시면 됩니다.
- **주의사항 :** 이 컴포넌트는 자체적으로 직사각형 모양의 메시를 생성하여 MeshFilter에 적용시킵니다. 

## 데모 버전 실행방법 
첨부된 파일의 압축을 풀고, 폴더의 Unity Portfolio Water.exe를 실행합니다.
